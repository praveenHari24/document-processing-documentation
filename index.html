Enhance PDF documents with a range of capabilities. Edit PDFs by merging, splitting, rotating, deleting pages, flattening, and compressing files.

Split the PDF document.

Divide individual PDF files into multiple documents using various criteria, such as page range, file count, or page count.

Authorizations:
Bearer
Request Body schema: multipart/form-data
File
required
string

Specifies the input file to split.

Password
string

Specifies the password to unprotect the input file.

SplitOption.FileCount
integer <int32>

Specifies the number of files to split the input file into.

SplitOption.PageCount
integer <int32>

Specifies the number of pages to split the input file into.

Array of objects (SplitRange)

Specifies the page ranges to split the input file into.

Responses

/v1/edit-pdf/split

Convert

The Syncfusion document processing engine can handle various file formats as input, including Word, Excel, PowerPoint, and HTML, converting them into PDF documents.

Converts a Word document to a PDF.

Converting a Word document to a PDF is easy. It supports various formats like .doc, .docx, and .rtf. You can adjust settings, such as accessibility and archiving, to suit your needs.

Authorizations:
Bearer
Request Body schema: multipart/form-data
File
required
string

The input Word document.

Password
string

Specifies the password to open the protected input Word document.

PreserveFormFields
boolean

Convert the word document with form fields.

PdfComplaince
string

Convert the word document with PDF/A compliance. Supported values are PDF/A-1B, PDF/A-2B, PDF/A-3B and PDF/A-4.

EnableAccessibility
boolean

Create accessibility tags in the PDF document.

Responses

/v1/conversion/word-to-pdf

Request samples

  • Payload
Content type
multipart/form-data
{
  "file": "invoice.docx",
  "password": "password",
  "preserveFormFields": true,
  "pdfComplaince": "PDF/A-1B",
  "enableAccessibility": true
}

Converting HTML to PDF is straightforward and preserves all elements such as graphics, images, text, fonts, and layout from the original HTML document.

Bearer
Request Body schema: multipart/form-data
IndexFile
string

Url
string

Url of the web page to convert to PDF

Assets
Array of strings

List of assets to include in the conversion

Margin
integer <int32>

AdditionalDelay
integer <int32>

Additional delay to wait before converting the page to PDF

ViewPortWidth
